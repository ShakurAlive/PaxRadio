# Logo Directory

Drop your PNG or JPG station logos here.

Filename format should match the third column in radio.list

Example:
- rock_fm.png
- jazz_radio.png
- news_24.png

Recommended size: 512x512 pixels or larger
Supported formats: PNG, JPG, JPEG
